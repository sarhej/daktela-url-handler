### Prerequisites
Make sure Daktela is installed. Download it from [daktela.com](https://www.daktela.com/)

### Step 1: Run the Installer
1. Find: **`Install-Daktela-Handler.vbs`**
2. **Double-click it**
3. If prompted by Windows, click: **"Yes"** to allow admin access
4. Wait for the message to appear

### Step 2: One-Time Setup (First Time Only)
When you click a phone link:
1. Windows shows a list of apps (this is normal)
2. **Select "Daktela"** from the list
3. **Check the box** "Always use this app"
4. Click **OK**

After that, phone links will **always open directly in Daktela**.

## â“ FAQ

### Q: Why do I see a dialog?
**A:** Windows protects you from malware by requiring your permission before changing app handlers. Click through once, and it's done forever.

### Q: Do I need admin rights?
**A:** Yes, only for installation (one time). Daily use doesn't need admin.

### Q: Is it safe?
**A:** Yes! It just tells Windows to use Daktela for phone links. Completely safe.

### Q: What if I get an error?
**A:** Run the installer again with "Run as Administrator".

### Q: Can I undo it?
**A:** Yes, but it requires IT help. Contact your IT admin if needed.

### Q: What if a different app opens?
**A:** Another app might have been set as default. Just run the installer again.

### Q: Can I test it without clicking a real number?
**A:** Yes! Use `test-links.html` which has test numbers that won't actually dial.

## ğŸ”§ Troubleshooting

### Nothing happens when I double-click the installer
- Make sure you have admin rights
- Right-click the file
- Select "Run as Administrator"

### "Access Denied" when running
- Right-click: `Install-Daktela-Handler.vbs`
- Select: "Run as Administrator"

### "Daktela not found" error
- Make sure Daktela is installed first
- Restart your computer and try again
- Contact IT if problems persist

### Links open in wrong app
- Daktela might not be installed
- Another app might have priority
- Run the installer again

### Still seeing the dialog every time
- This is normal the first time
- Just select Daktela and check "Always use this app"
- If it keeps showing after that, contact IT
